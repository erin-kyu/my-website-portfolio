## ‚ö° Fast Track to GitHub Pages (5 minutes)

### Step 1: Extract the Files
1. Unzip `my-website.zip`
2. You'll get a folder called `my-website`

### Step 2: Add Your Resume Photo (Optional but Recommended)
1. Add a photo named `resumepic.jpg` to the `img/` folder
2. OR edit `resume.html` line 12 to use `img/picofme.jpg` instead

### Step 3: Open in VS Code
1. Open Visual Studio Code
2. Click `File` ‚Üí `Open Folder`
3. Select the `my-website` folder

### Step 4: Test Locally (Recommended)
1. Install "Live Server" extension in VS Code
2. Right-click `index.html`
3. Select "Open with Live Server"
4. Your website opens in browser - check everything works!

### Step 5: Push to GitHub
1. In VS Code, click the **Source Control** icon (left sidebar)
2. Click **Initialize Repository**
3. Click the **+** icon to stage all files
4. Type commit message: `Initial website upload`
5. Click **Commit**
6. Click **Publish Branch**
7. Sign in to GitHub if prompted
8. Choose:
   - Repository name: `my-website`
   - Visibility: **Public** ‚Üê Important!
9. Wait for upload to complete

### Step 6: Enable GitHub Pages
1. Go to your GitHub repository in browser
2. Click **Settings** tab
3. Click **Pages** in left sidebar
4. Under "Source":
   - Branch: `main`
   - Folder: `/(root)`
5. Click **Save**
6. Wait 1-2 minutes

### Step 7: Access Your Live Website! üéâ
Your website is now live at:
```
https://YOUR-USERNAME.github.io/my-website/
```

Replace `YOUR-USERNAME` with your actual GitHub username.

## ‚ö†Ô∏è Common Issues

**"My images aren't showing!"**
- Check all images are in `img/` folder
- Check file names match exactly (case-sensitive!)
- For resume photo: add `resumepic.jpg` OR use `picofme.jpg`

**"GitHub Pages shows 404 error"**
- Wait 2-3 minutes (it takes time to deploy)
- Check Settings ‚Üí Pages shows green checkmark
- Verify branch is `main` and folder is `/(root)`
- Make sure repository is Public

**"My CSS isn't working"**
- Check all CSS files are in `css/` folder
- Clear browser cache: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

**"Resume link doesn't work"**
- Check `menu.html` has the resume link
- Check `resume.html` exists in root folder
- Refresh the browser
